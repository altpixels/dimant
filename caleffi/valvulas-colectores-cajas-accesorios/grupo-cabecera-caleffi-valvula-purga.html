Grupo de cabecera Caleffi con válvula de purga

El grupo terminal Caleffi se utiliza en los colectores de las instalaciones de calefacción, tanto para purga como para llenado y drenado de los circuitos.

Descripción

El grupo de cabecera con válvula de purga de aire manual y preparación para grifo de descarga se utiliza en los colectores para purgar el aire de la instalación, así como para llenado y drenado de cada uno de los circuitos de dicho colector.

Funcionamiento

El grupo de cabecera para colector dispone de una perilla plástica que se debe girar en forma manual para eliminar el aire de la instalación, esta perilla se puede sustituir por una válvula eliminadora de aire para que esta expulsión de aire sea de forma automática. Además, está provisto de una salida para drenar o llenar el sistema de agua en la cual tiene un tapón, pero éste se puede sustituir también por un grifo de descarga con válvula de cierre, adaptador para manguera y tapón.

Características principales

  • Cuenta con racores de doble conexión radial y tapón.
  • Presión máxima de servicio: 10 bar.
  • Campo de temperatura: 5–100 °C.
